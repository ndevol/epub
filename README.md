# Convert HTML Files to EPUBs

## HTML 

Prepare an HTML with the desired content in the body:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Book</title>
    <link rel="stylesheet" href="../../style/main.css">
</head>
<body>
</body>
</html>
